
# Fine-Tuning Llama-2-7B with QLoRA

This repository contains the code to fine-tune the `NousResearch/Llama-2-7b-chat-hf` model.

## ⚙️ Setup and Installation
1. Clone the repository: `git clone https://github.com/your-github-username/your-repo-name.git`
2. Install requirements: `pip install -r requirements.txt`

## ▶️ How to Run
To start the fine-tuning, run the script: `python train.py`
